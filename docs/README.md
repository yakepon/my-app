# FEELCYCLE LOG

FEELCYCLEのライド記録をGoogleスプレッドシートに保存・一覧表示するアプリです。

## セットアップ手順

1. Googleスプレッドシートを新規作成する
2. メニューから「拡張機能」→「Apps Script」を開く
3. デフォルトの `Code.gs` の中身を、このフォルダ内の `gas/Code.gs` の内容で置き換えて保存する
4. 右上の「デプロイ」→「新しいデプロイ」をクリック
   - 種類の選択: 「ウェブアプリ」
   - 実行するユーザー: 自分
   - アクセスできるユーザー: 全員
5. 「デプロイ」をクリックし、表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/.../exec`）をコピーする
6. `index.html` をブラウザで開き、「接続設定」欄に手順5のURLを貼り付けて「保存」をクリックする

これで記録の追加・一覧表示がスプレッドシートと連携します。

## 補足

- スプレッドシートには初回アクセス時に自動で `records` シートとヘッダー行（`id, datetime, studio, bikeNo, category, program, instructor, calories, memo`）が作成されます。
- 記録の編集・削除に対応しています。一覧の各カードの「編集」「削除」ボタンから操作してください。
- 既存の `records` シートを使っている場合も、列はヘッダー名で識別されるため既存データの列順はそのまま使えます。`bikeNo` などこのバージョンで追加された列が無い場合は、再デプロイ後の初回アクセス時にヘッダー行の末尾へ自動で追加されます。
- コードを変更した場合は、Apps Scriptの「デプロイを管理」から新しいバージョンを再デプロイしてください。

## ロゴ画像の設定

ヘッダーに表示するロゴ画像は `docs/assets/feelcycle-log.jpg` に配置してください。画像が無い場合はロゴ部分は非表示になり、テキストロゴのみ表示されます。
